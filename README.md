# PredictionModel-SARIMA
# This respitory has the a SARIMA model for predicting 30 days in advanced
#The purpose of this time-series forecasting model is to predict daily revenue trends using daily transactional data.
#The script produces visual forecast outputs and translates results into business recommendations for inventory and financial planning.
#Cleaned and transformed raw transaction data by removing returns and engineering revenue features, improving signal quality for modeling.
#Implemented 80/20 time-series validation and evaluated model performance using RMSE and MAPE metrics. Identified impact of outliers and assessed their influence on forecast accuracy to guide model improvement strategies
